## Available Scripts

In the project directory, you can run:

### `node server`

Runs the app in the development mode.<br />

### `node server -h`

Runs the app in the development help mode.<br />

### `node server list`

all contact show<br />

### `node server read`

single contact view.<br />

### `node server add`

to add single contact<br />

### `node server delete`

delete contact .<br />
